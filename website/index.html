Online banking APPLICATION


What the project is about?


Build a banking application that is easy to use, secure and boasts good up-time.
Users can create account simply by giving their basic details.
A registered user can transact with other registered users, add or withdraw credit
and view their recent transactions.
Users will be alerted with email and text message for new transactions.
Users also have the option to change their details easily.


Details


C++ is used as the main language.
Qt for C++ for creating user interface.
Mysql database for storing data.
Microsoft Azure for hosting MySql database.
Python for all communications.
MD5 Hashing for passwords.
